This site is built with [Zola](https://getzola.org/),
a static site generator that uses [Tera](https://keats.github.io/tera/) template engine.
The theme is [Hook](https://getzola.org/themes/hook/) and the font is [Iosevka](https://typeof.net/Iosevka/).

It does not collect user data and does not use cookies, analytics, or tracking.
It is hosted at [statichost.eu](https://statichost.eu), a privacy-conscious European provider.
Deployments are automated through webhooks.

The source code is avalaible on [GitHub](https://github.com/solargc/buyse-dev).
